# vacation-planner-apk
Vacation Planner - D424
Stacey Crippen

This repository hosts the official APK file for the Vacation Planner Android application, created as part of the WGU Software Engineering Capstone (D424). 

App Features include:
*Add, edit, and delete vacations
*Manage excursions for each vacation
*Vacation report generator (with date range + timestamp)
